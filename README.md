# Habitual (Work in progress)

A simple ranged-habit tracker app that allows you find accountability partners.

## How to run
In your terminal;
- `git clone https://github.com/MikeyOnyedika/habitual.git`
- `cd ./habitual/frontend`
- `npm i`
- `npm run dev`

## Frontend routes
- `/login`
- `/signup`
- `/overview`
